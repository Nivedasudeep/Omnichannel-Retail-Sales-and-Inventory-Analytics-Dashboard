Omnichannel Retail Sales & Inventory Analytics Dashboard
Project Overview

This project focuses on analyzing retail sales data from multiple locations to uncover sales trends, customer purchasing behavior, and inventory insights. The dashboard was built using Python, PostgreSQL, and Power BI to help retail managers make data-driven business decisions.

The project includes:

Data cleaning and preprocessing
SQL-based business analysis
Interactive Power BI dashboard
KPI tracking and visualization
Business Problem

Retail businesses often struggle to:

Track sales performance across regions
Identify top-selling products
Understand seasonal sales trends
Monitor profitability and inventory movement

This dashboard provides a centralized analytics solution for monitoring retail performance.
